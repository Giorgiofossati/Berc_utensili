# SUPABASE_SCHEMA

## Actual Database Structure

The following are the fields in the current database structure:

- **tipologia**: Type of tool
- **forma**: Shape of tool
- **diametro**: Diameter of tool
- **diametro_nominale**: Nominal diameter
- **raggio**: Radius
- **passo**: Pitch
- **tolleranza**: Tolerance
- **lunghezza**: Length
- **rotazione**: Rotation
- **materiale**: Material
- **rivestimento**: Coating
- **stato**: Status
- **ubicazione**: Location
- **progetto**: Project
- **codice**: Code
- **serial_number**: Serial Number
- **fornitore**: Supplier
- **quantita**: Quantity
- **lavorazione**: Processing
- **sistema_misura**: Measurement system
- **alias**: Alias
